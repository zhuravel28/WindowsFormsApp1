# Менеджер турів

## Опис проєкту

Це Windows Forms програма для роботи з туристичними замовленнями.  
Програма дозволяє переглядати, додавати та видаляти замовлення турів.

## Основні можливості

- перегляд туристичних замовлень;
- перехід між замовленнями кнопками вперед і назад;
- додавання нового замовлення;
- видалення поточного замовлення;
- розрахунок загальної вартості туру;
- підтримка двох типів турів: звичайний та гарячий.

## Типи турів

### Звичайний тур

Звичайний тур розраховується без знижки:

Кількість × Ціна за 1

### Гарячий тур

Гарячий тур має знижку у відсотках.  
Програма спочатку рахує загальну суму, а потім віднімає знижку.

## Використані технології

- C#
- Windows Forms
- .NET Framework
- Visual Studio

## Структура проєкту

- `Program.cs` — запуск програми;
- `Form1.cs` — основна логіка форми;
- `Form1.Designer.cs` — елементи інтерфейсу;
- `Tours.cs` — класи для роботи з туристичними замовленнями.

## Як запустити проєкт

1. Відкрити файл `WindowsFormsApp1.sln` у Visual Studio.
2. Натиснути кнопку `Пуск` або клавішу `F5`.
3. Після запуску відкриється вікно програми "Менеджер Турів".

## Демонстрація роботи

У програмі можна заповнити дані замовлення: код, ПІБ клієнта, назву туру, кількість, ціну та знижку.  
Після натискання кнопки "Додати замовлення" запис додається до списку.  
Кнопки `<` і `>` дозволяють переглядати замовлення.  
Кнопка "Видалити поточне" видаляє вибране замовлення.
